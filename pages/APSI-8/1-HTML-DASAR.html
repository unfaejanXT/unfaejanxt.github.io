1. Apa itu HTML?

HTML (HyperText Markup Language) bukan bahasa pemrograman. Ini bahasa markup untuk memberi struktur pada konten web.

Prinsip inti: Tag membuka dan menutup memberi makna. Browser membaca makna itu, bukan tampilannya. 
<p>Ini paragraf</p>
<h1>Ini judul utama</h1>

2. Struktur Dokumen Wajib

Setiap file .html harus dimulai seperti ini. Hafalkan.

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul di Tab Browser</title>
</head>
<body>
  <!-- Semua yang terlihat user ada di sini -->
</body>
</html>
  • DOCTYPE: mode HTML5
  • lang="id": bantu screen reader dan SEO
  • charset UTF-8: agar karakter Indonesia tidak rusak
  • viewport: wajib untuk mobile

3. Teks, Heading, dan Formatting

Heading: H1 sampai H6

<h1>Judul Halaman (1x saja)</h1>
<h2>Sub Bab</h2>
<h3>Sub-sub</h3>

Gunakan berurutan. Jangan loncat dari h1 ke h3.

Paragraf dan Format

<p>Paragraf biasa.</p>
<strong>Tebal penting</strong>
<em>Miring penekanan</em>
<br> <!-- ganti baris -->
<hr> <!-- garis horizontal -->

5. Link dan Navigasi

<a href="https://example.com">Kunjungi</a>
<a href="tentang.html">Halaman internal</a>
<a href="#bagian2">Lompat ke bagian</a>
<a href="mailto:email@kampus.ac.id">Kirim email</a>
<a href="tel:+628123456">Telepon</a>

Wajib: selalu isi href. Untuk buka tab baru: target="_blank" rel="noopener"

6. Gambar

<img src="gambar.jpg" alt="Deskripsi gambar untuk aksesibilitas" width="300">
  • alt wajib diisi, bukan opsional
  • Gunakan figure dan figcaption untuk gambar dengan keterangan
<figure>
  <img src="logo.png" alt="Logo kampus">
  <figcaption>Logo Universitas 2026</figcaption>
</figure>

7. Tabel

Gunakan hanya untuk data tabular, bukan layout.

<table border="1">
  <thead>
    <tr><th>NIM</th><th>Nama</th></tr>
  </thead>
  <tbody>
    <tr><td>001</td><td>Ani</td></tr>
    <tr><td>002</td><td>Budi</td></tr>
  </tbody>
</table>

8. Form — Materi Paling Penting

Form adalah interaksi. Kuasai ini, kalian bisa buat login, register, kontak.

<form action="/kirim" method="POST">
  <label for="nama">Nama Lengkap</label>
  <input type="text" id="nama" name="nama" required placeholder="Masukkan nama">

  <label for="email">Email</label>
  <input type="email" id="email" name="email" required>

  <label for="password">Password</label>
  <input type="password" id="password" name="password" minlength="8" required>

  <label for="jurusan">Jurusan</label>
  <select id="jurusan" name="jurusan">
    <option value="ti">Teknik Informatika</option>
    <option value="si">Sistem Informasi</option>
  </select>

  <p>Jenis Kelamin:</p>
  <input type="radio" id="l" name="jk" value="L"> <label for="l">Laki-laki</label>
  <input type="radio" id="p" name="jk" value="P"> <label for="p">Perempuan</label>

  <input type="checkbox" id="setuju" name="setuju" required>
  <label for="setuju">Saya setuju</label>

  <label for="pesan">Pesan</label>
  <textarea id="pesan" name="pesan" rows="4"></textarea>

  <button type="submit">Kirim</button>
</form>
Aturan emas: label for harus sama dengan id input. Ini aksesibilitas.

9. Semantic HTML5 — Wajib 2026

Berhenti pakai div semua. Gunakan tag bermakna:

<header>Logo dan navigasi</header>
<nav>Menu link</nav>
<main>
  <article>
    <h1>Judul Artikel</h1>
    <section>Bagian 1</section>
    <section>Bagian 2</section>
  </article>
  <aside>Sidebar terkait</aside>
</main>
<footer>Copyright</footer>

Manfaat: SEO naik, screen reader paham struktur, kode lebih bersih.

12. Checklist Materi Wajib Dikuasai

Struktur
  • DOCTYPE, html, head, body
  • meta charset dan viewport
  • title yang deskriptif
Konten
  • h1-h6 berurutan
  • p, strong, em
  • ul, ol, li
Media & Link
  • a href dengan target
  • img dengan alt wajib
  • figure dan figcaption
Form & Tabel
  • form, label, input semua type
  • select, textarea, button
  • table, thead, tbody, th, td
Semantic

header, nav, main, article, section, aside, footer — ganti kebiasaan pakai div.

Untuk besok: Minta mahasiswa buat 1 halaman biodata menggunakan semua tag di atas tanpa CSS. Jika bisa, mereka sudah paham HTML.
