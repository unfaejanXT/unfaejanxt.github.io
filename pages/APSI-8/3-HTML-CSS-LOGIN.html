Belajar HTML & CSS dari Nol lewat Form Login

Panduan lengkap untuk mahasiswa. Tanpa playground, tanpa kuis — hanya materi terstruktur dan implementasi langkah demi langkah.

Besok kita akan bedah dua file: login.html dan register.html.

1. Pendahuluan

Tujuan Pembelajaran
  • Memahami struktur dokumen HTML5 yang benar
  • Menguasai CSS dasar: reset, box model, flexbox, dan styling form
  • Mampu membangun halaman login dan register dari nol dengan kode bersih
  • Belajar pola penulisan CSS yang rapi dan mudah dirawat

Kenapa mulai dari form login? Karena form menggabungkan hampir semua konsep dasar: struktur HTML, layout CSS, tipografi, warna, dan interaksi hover. Setelah ini, kalian bisa membuat komponen apapun.

2. HTML Dasar

2.1 Kerangka Wajib HTML5

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <!-- konten di sini -->
</body>
</html>

Penjelasan: DOCTYPE memberi tahu browser ini HTML5. Meta viewport penting agar responsif di HP. Semua konten visual harus di dalam body.

2.2 Tag Penting untuk Form

TagFungsiCatatan
<form>Wadah inputaction="" method="POST" untuk kirim data
<label for="id">Label aksesibelfor harus sama dengan id input
<input type="text/password">Kolom isianSelalu pakai placeholder dan required
<button type="submit">Tombol aksiBedakan dengan type="button"
<div class="form-group">PembungkusBukan tag bawaan, tapi pola untuk grouping

2.3 Semantik dan Aksesibilitas

Gunakan label, bukan hanya placeholder. Placeholder hilang saat mengetik. Label tetap terbaca screen reader. Ini membedakan kode amatir dan profesional.

3. CSS Dasar

3.1 Tiga Cara Menulis CSS

Untuk materi ini kita pakai internal CSS di dalam <style> agar 1 file. Di proyek nyata, pisahkan ke file .css.

3.2 Reset Dasar

* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}

box-sizing: border-box membuat width termasuk padding dan border. Tanpa ini, layout form akan meluber.

3.3 Layout dengan Flexbox

body {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

Ini trik klasik untuk membuat kartu login di tengah layar secara vertikal dan horizontal. 100vh = 100% tinggi viewport.

3.4 Box Model

Setiap elemen punya: content → padding → border → margin. Di login-box kita pakai:

  • max-width: 320px agar tidak terlalu lebar di desktop
  • padding: 30px ruang napas di dalam kartu
  • border: 1px solid #ccc dan border-radius: 8px untuk kesan modern

3.5 Pola Penulisan Rapi

Lihat di file contoh, CSS dikelompokkan dengan komentar: /* 1. RESET */, /* 2. LAYOUT */, dst. Dan properti diurutkan: Layout dulu, baru Text. Ini kebiasaan bagus untuk tim.

4. Praktik Langkah demi Langkah: Login

Kita akan membangun persis seperti file login.html.

<div class="login-box">
  <h2>Login Gudang</h2>
  <form action="" method="POST">
    <div class="form-group">
      <label for="username">Username</label>
      <input type="text" id="username" name="username" placeholder="Masukkan username" required>
    </div>

</div>

5. Praktik: Register

Register hampir identik. Bedanya:

  • Tambah field Nama Lengkap di atas username
  • Ganti judul jadi "Daftar Akun"
  • Ubah warna tombol menjadi hijau (#2ecc71) untuk membedakan aksi
  • Hover hijau lebih gelap (#27ae60)

Ini mengajarkan prinsip reuse. Kita pakai class .login-box yang sama, hanya ubah konten dan warna.

6. Best Practice untuk Mahasiswa

1. Konsistensi

Gunakan pola komentar dan urutan properti yang sama di semua project. Ini mempercepat debugging.

2. Mobile First

max-width bukan width fixed. Form akan otomatis mengecil di HP.

3. Aksesibilitas

Selalu pasangkan label dan id. Jangan andalkan placeholder saja.

4. Jangan Copy-Paste Buta

Pahami kenapa pakai flex, kenapa box-sizing. Besok saat ujian praktik, kalian harus bisa tulis dari ingatan.

7. Penutup dan Tugas

Besok di kelas:

  1. Ketik ulang login.html tanpa melihat, 15 menit
  2. Modifikasi menjadi register dengan tambahan field email
  3. Ubah skema warna menjadi tema kampus kalian
Ingat: HTML adalah struktur, CSS adalah presentasi. Pisahkan logikanya di kepala, maka kodenya akan bersih.
