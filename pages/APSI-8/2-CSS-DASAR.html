Belajar CSS dari Nol — Materi yang Wajib Dikuasai

Pahami konsep, bukan hafal properti. Setelah ini kalian bisa styling apapun tanpa Bootstrap.

1. Apa itu CSS?

CSS (Cascading Style Sheets) mengatur tampilan. HTML = struktur, CSS = presentasi.

Prinsip inti: Selector menunjuk elemen, Declaration (properti: nilai) mengatur gayanya. 
p {
  color: blue;
  font-size: 16px;
}

2. Tiga Cara Menyisipkan CSS

Inline
<p style="color:red">
Internal
<style> p{...} </style>
External (Wajib)
<link rel="stylesheet" href="style.css">

Untuk proyek nyata: selalu pakai external. Untuk belajar besok, internal cukup.

3. Selector Dasar — Wajib Hafal

/* elemen */
p { margin: 0; }

/* class - paling sering dipakai */
.card { padding: 20px; }

/* id - hanya 1x per halaman */
#header { background: black; }

/* universal */
* { box-sizing: border-box; }

/* grup */
h1, h2, h3 { font-family: Arial; }

/* descendant */
nav a { text-decoration: none; }

/* child langsung */
ul > li { list-style: none; }

4. Cascade, Specificity, Inheritance

Cascade: yang ditulis terakhir menang jika specificity sama.

Specificity urutan: inline (1000) > id (100) > class (10) > elemen (1)

/* skor 1 */
p { color: black; }

/* skor 10 - menang */
.text { color: blue; }

/* skor 100 - menang */
#judul { color: red; }

Inheritance: properti seperti color dan font diturunkan ke anak. Margin tidak.

5. Warna dan Satuan

Warna
color: red;
color: #ff0000;
color: #f00;
color: rgb(255,0,0);
color: rgba(255,0,0,0.5);
Satuan
  • px — absolut
  • % — relatif ke parent
  • em — relatif ke font parent
  • rem — relatif ke root (16px)
  • vh/vw — 1% tinggi/lebar layar

6. Box Model — Inti CSS

Setiap elemen = content + padding + border + margin.

CONTENT
.box {
  width: 200px;      /* content */
  padding: 20px;     /* ruang dalam */
  border: 5px solid blue;
  margin: 30px;      /* ruang luar */
}

Wajib pakai di awal: * { box-sizing: border-box; } agar width termasuk padding dan border.

7. Display

display: block;       /* p, div - ambil 1 baris penuh */
display: inline;      /* span, a - selebar konten */
display: inline-block; /* bisa atur width tapi tetap sebaris */
display: none;        /* hilang total */
display: flex;        /* aktifkan flexbox */

8. Position

position: static;   /* default */
position: relative; /* geser dari posisi normal */
  top: 10px; left: 20px;
position: absolute; /* relatif ke parent yang relative */
position: fixed;    /* nempel layar - untuk navbar */
position: sticky;   /* nempel saat scroll */

9. Flexbox — Layout Modern

Untuk 1 dimensi (baris atau kolom). Wajib kuasai.

.container {
  display: flex;
  justify-content: center; /* horizontal: flex-start, center, space-between */
  align-items: center;     /* vertikal: flex-start, center */
  gap: 20px;               /* jarak antar item */
  flex-wrap: wrap;         /* turun baris jika penuh */
}

.item {
  flex: 1; /* bagi rata */
}

10. CSS Grid — Layout 2 Dimensi

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 kolom sama */
  gap: 15px;
}

/* atau responsif otomatis */
.grid-auto {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

11. Typography

body {
  font-family: 'Segoe UI', Arial, sans-serif;
  font-size: 16px;
  line-height: 1.6; /* jarak baris */
  font-weight: 400; /* 400 normal, 700 bold */
  color: #333;
}

h1 {
  font-size: 2rem; /* 32px */
  letter-spacing: -0.5px;
  text-align: center;
}

12. Background, Border, Shadow

.card {
  background-color: #fff;
  background-image: url('bg.png');
  background-size: cover;
  border: 1px solid #ddd;
  border-radius: 8px; /* sudut tumpul */
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

13. Pseudo-class dan Pseudo-element

/* saat interaksi */
a:hover { color: red; }
button:active { transform: scale(0.98); }
input:focus { border-color: blue; outline: none; }

/* struktur */
li:first-child { font-weight: bold; }
li:nth-child(odd) { background: #f5f5f5; }

/* pseudo-element */
p::before { content: "→ "; }
p::after { content: " ✓"; }

14. Responsive Dasar

/* mobile first */
.container { width: 100%; padding: 15px; }

/* tablet ke atas */
@media (min-width: 768px) {
  .container { max-width: 720px; margin: auto; }
}

/* desktop */
@media (min-width: 1024px) {
  .container { max-width: 960px; }
}

Gunakan viewport meta di HTML: <meta name="viewport" content="width=device-width, initial-scale=1.0">
