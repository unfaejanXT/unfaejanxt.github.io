Evolusi Kode: Membedah Form Login

Dalam Analisis dan Perancangan Sistem Informasi (APSI), antarmuka adalah jembatan antara user dan sistem. Kita akan membedah bagaimana elemen mentah HTML berubah menjadi UI profesional.

2. Bedah Struktur HTML (The Skeleton)

Berikut adalah kode HTML murni untuk form login tanpa gaya sedikitpun. Pahami peran setiap barisnya.

<!-- Pembungkus Utama -->
<div class="login-box">
    <h2>Login Sistem</h2>
    
    <form>
        <!-- Grup Input Username -->
        <div class="form-group">
            <label for="user">Username</label>
            <input type="text" id="user" required>
        </div>

        <!-- Grup Input Password -->
        <div class="form-group">
            <label for="pass">Password</label>
            <input type="password" id="pass" required>
        </div>

        <!-- Checkbox Ingat Saya -->
        <div class="form-check">
            <input type="checkbox" id="remember">
            <label for="remember">Ingat Saya</label>
        </div>

        <button type="submit">Masuk</button>
    </form>
</div>
ElemenFungsi Analitis
<form>Wadah besar untuk membungkus semua inputan (seperti username dan password) agar bisa dikirim bersamaan.
<div class="login-box">Kotak pembungkus agar tampilan form login mengumpul menjadi satu kartu yang rapi.
<label for="...">Teks judul di atas kolom input. Jika judul ini diklik, kursor otomatis masuk ke kolom input pasangannya.
<input type="text">Kotak kosong yang bisa diketik dengan tulisan biasa (untuk mengisi Username).
<input type="password">Kolom input khusus yang otomatis menyembunyikan ketikan kita menjadi titik-titik bulat agar aman.
<input type="checkbox">Kotak centang kecil untuk pilihan Ya/Tidak (misalnya untuk centang "Ingat Saya").
<button type="submit">Tombol utama untuk mengirim data login atau masuk ke dalam aplikasi.
required (Atribut)Tanda wajib isi. Kolom input tidak boleh dikosongkan saat tombol masuk diklik.

3. Tampilan Sementara (HTML Saja)

Tanpa CSS, browser akan merender elemen secara berurutan ke bawah (flow) dengan gaya default yang sangat sederhana.

*Inilah tampilan asli web sebelum diberikan 'sentuhan visual'.

4. Step-by-Step Penambahan CSS

Step 1 Reset & Centering (Flexbox)

Tujuannya membuang margin bawaan browser dan meletakkan form tepat di tengah layar. Flexbox sangat efektif untuk layouting elemen.

Properti CSSDeskripsi Fungsi
box-sizing: border-box; Membuat ukuran elemen tetap sesuai ukuran yang ditentukan meskipun diberi padding atau border.
margin: 0; Menghapus jarak luar bawaan browser agar layout lebih rapi dan konsisten.
padding: 0; Menghapus jarak dalam bawaan browser pada semua elemen.
display: flex; Mengaktifkan sistem layout Flexbox agar posisi elemen lebih mudah diatur.
justify-content: center; Memposisikan isi tepat di tengah secara horizontal.
align-items: center; Memposisikan isi tepat di tengah secara vertikal.
height: 100vh; Membuat tinggi halaman memenuhi seluruh tinggi layar perangkat.
background-color: #f2f2f2; Memberikan warna latar abu-abu terang pada halaman.
font-family: sans-serif; Mengatur jenis huruf menjadi sederhana dan mudah dibaca. 
* {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
}
body {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
    background-color: #f2f2f2;
    font-family: sans-serif;
}
Step 2 Membuat Kartu (The Container)

Memberikan batas visual berupa garis tepi (border) tipis dan warna putih agar form terlihat terstruktur dan terpisah dari latar belakang.

Properti CSSDeskripsi Fungsi
width: 100%; Membuat kotak login fleksibel mengikuti ukuran layar.
max-width: 320px; Membatasi lebar maksimum form agar tidak terlalu besar.
padding: 30px; Memberikan ruang di dalam kotak agar isi tidak menempel ke tepi.
background: white; Memberikan warna putih pada kartu login.
border-radius: 8px; Membuat sudut kotak menjadi melengkung.
border: 1px solid #ccc; Menambahkan garis tipis abu-abu di sekeliling form.
text-align: center; Membuat teks judul berada di tengah.
margin-bottom: 20px; Memberikan jarak bawah pada judul sebelum form dimulai.
color: #333; Mengubah warna teks menjadi abu gelap agar nyaman dibaca. 
.login-box {
    width: 100%;
    max-width: 320px;
    padding: 30px;
    background: white;
    border-radius: 8px;
    border: 1px solid #ccc;
}
h2 {
    text-align: center;
    margin-bottom: 20px;
    color: #333;
}
Step 3 Styling Input & Label

Memberikan ruang (padding) pada input dan mengatur label agar diletakkan dengan rapi di atas kolom isian, bukan di sebelahnya.

Properti CSSDeskripsi Fungsi
margin-bottom: 15px; Memberikan jarak antar grup input agar tidak terlalu rapat.
display: block; Membuat label berada di baris sendiri di atas input.
margin-bottom: 5px; Memberikan sedikit jarak antara label dan kolom input.
font-size: 14px; Mengatur ukuran teks label menjadi lebih kecil dan rapi.
width: 100%; Membuat input memenuhi lebar kotak login.
padding: 10px; Memberikan ruang di dalam input agar nyaman saat mengetik.
border: 1px solid #ddd; Menambahkan garis tipis pada kolom input.
border-radius: 4px; Membuat sudut input sedikit melengkung.
display: flex; Membuat checkbox dan label sejajar dalam satu baris.
align-items: center; Menyelaraskan posisi checkbox dan teks secara vertikal.
gap: 8px; Memberikan jarak antara checkbox dan teks label. 
.form-group {
    margin-bottom: 15px;
}
.form-group label {
    display: block;
    margin-bottom: 5px;
    font-size: 14px;
}
.form-group input {
    width: 100%;
    padding: 10px;
    border: 1px solid #ddd;
    border-radius: 4px;
}
.form-check {
    display: flex;
    align-items: center;
    gap: 8px;
    margin-bottom: 15px;
}
.form-check label {
    font-size: 14px;
    margin-bottom: 0;
}
Step 4 Final: Tombol & Interaksi

Langkah terakhir adalah memberikan warna pada tombol (Call-to-Action) dan menyematkan efek hover sebagai respon interaktif sistem.

Properti CSSDeskripsi Fungsi
width: 100%; Membuat tombol memenuhi lebar form login.
padding: 10px; Memberikan ruang di dalam tombol agar lebih nyaman diklik.
background: #2563eb; Memberikan warna biru pada tombol.
color: white; Mengubah warna tulisan tombol menjadi putih.
border: none; Menghilangkan garis bawaan browser pada tombol.
border-radius: 4px; Membuat sudut tombol sedikit melengkung.
cursor: pointer; Mengubah kursor menjadi ikon tangan saat tombol disentuh.
font-weight: bold; Membuat tulisan tombol menjadi lebih tebal.
button:hover Memberikan efek perubahan warna saat mouse diarahkan ke tombol.
background: #1d4ed8; Mengubah warna tombol menjadi lebih gelap saat hover. 
button {
    width: 100%;
    padding: 10px;
    background: #2563eb;
    color: white;
    border: none;
    border-radius: 4px;
    cursor: pointer;
    font-weight: bold;
}
button:hover {
    background: #1d4ed8;
}
